# Expectation Maximization over Fourier Series (emofs)

## Overview

This repository provides experimental MATLAB code in draft form. The implementation is intended for research and exploratory purposes and may be updated or modified over time.

## Requirements

The code has been tested with:

* MATLAB R2025b

Compatibility with other MATLAB versions is not guaranteed.

## Usage

To run the code:

1. Open MATLAB
2. Navigate to the project directory
3. Open any `.m` file starting with `comparison`
4. Run the script

## Notes

* The code is provided in an early-stage (draft) version and may contain errors or incomplete features.
* Functionality and structure may change without notice.
